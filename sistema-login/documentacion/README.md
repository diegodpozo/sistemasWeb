# SISTEMA DE LOGIN SEGURO EN PHP

ESTE ES UN SISTEMA DE AUTENTICACION CONSTRUIDO CON PHP Y MYSQL, SIGUIENDO LAS MEJORES PRACTICAS DE SEGURIDAD.

## CARACTERISTICAS DE SEGURIDAD
- **PROTECCION CONTRA SQL INJECTION**: USO DE SENTENCIAS PREPARADAS CON PDO EN TODAS LAS CONSULTAS.
- **PROTECCION CONTRA XSS**: ESCAPE DE TODA SALIDA DE DATOS HACIA EL NAVEGADOR MEDIANTE HTMLSPECIALCHARS.
- **PROTECCION CONTRA CSRF**: GENERACION Y VALIDACION DE TOKENS UNICOS POR SESION EN FORMULARIOS POST.
- **GESTION DE CONTRASEÑAS**: ALMACENAMIENTO SEGURO USANDO BCRYPT MEDIANTE PASSWORD_HASH.
- **SEGURIDAD DE SESIONES**: REGENERACION DEL ID DE SESION AL LOGUEARSE Y CONFIGURACION DE COOKIES SEGURAS (HTTPONLY, SAMESITE=STRICT).
- **PROTECCION CONTRA FUERZA BRUTA**: LIMITE DE 5 INTENTOS FALLIDOS CON BLOQUEO TEMPORAL DE 15 MINUTOS.
- **CABECERAS DE SEGURIDAD**: IMPLEMENTACION DE CSP, X-FRAME-OPTIONS Y OTROS HEADERS DE PROTECCION.

## REQUISITOS
- SERVIDOR WEB (XAMPP, WAMP, ETC.) CON PHP 7.4 O SUPERIOR.
- BASE DE DATOS MYSQL.

## INSTALACION
1. IMPORTAR EL ARCHIVO `sql/esquema.sql` EN SU SERVIDOR DE BASE DE DATOS.
2. CONFIGURAR LAS CREDENCIALES EN `config/BaseDeDatos.php` SI ES NECESARIO.
3. ACCEDER A `registro.php` PARA CREAR UN NUEVO USUARIO.
4. INICIAR SESION DESDE `login.php`.
